# GitNote

> 用 Git 的方式，管理你的知识

GitNote 是一个基于 Git 的笔记平台，让知识管理更简单高效。

## 核心特性

### 🔄 Git 集成
- 无缝对接 GitHub/Gitee 仓库

### ✍️ 专业编辑
- 所见即所得的 Markdown 编辑器
- 支持数学公式、图表、代码高亮
- 简洁优雅的写作体验

### 🔌 拓展性强

- 基于插件架构，支持自定义插件

## 快速开始

```shell
pnpm i
pnpm dev
```

访问 http://localhost:5173/
