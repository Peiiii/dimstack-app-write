# GitNote

> 用 Git 的方式，管理你的知识

GitNote 是一个基于 Git 的笔记平台，让知识管理更简单高效。

## 核心特性

### ✍️ Typora & Notion 风格的编辑体验
- 所见即所得的 Markdown 编辑器
- Slash commands 快速格式化
- 支持数学公式、图表、代码高亮
- 直观优雅的写作体验

### 🔄 直连 Git 仓库
- 直接连接 GitHub/Gitee 仓库
- 无需中间存储，直接操作你的仓库

### 📂 更好的仓库阅读器
- 支持打开代码文件，语法高亮显示
- 比传统 Git 客户端更好的仓库浏览体验
- 轻松浏览你的代码库

### 🎨 VS Code 风格的界面
- 熟悉的侧边栏、编辑器和面板布局
- 可自定义的工作区
- 专业的开发环境

## 快速开始

```shell
pnpm i
pnpm dev
```

访问 http://localhost:5173/
